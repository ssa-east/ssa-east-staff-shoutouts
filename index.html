One-time Firebase Setup (~10 min)

Firebase gives every staff member real-time access to the same slide deck. It's free for schools at this scale.

1

Create a Firebase project

Go to firebase.google.com → "Get started" → "Create a project". Name it sonoran-shoutouts. Disable Google Analytics (not needed).

2

Create a Firestore database

In your project: Build → Firestore Database → Create database → Start in test mode → pick region us-central1 → Done.

3

Set Firestore rules

Firestore → Rules tab → replace everything with:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /shoutouts/{doc} { allow read, write: if true; } } }

Click Publish.

4

Get your Firebase config

Project Settings (gear icon) → Your apps → Web () → Register app → copy the firebaseConfig object.

5

Paste config into this file

Open sonoran-shoutouts.html in a text editor. Find the FIREBASE CONFIG section near the bottom and replace the placeholder values with your real config.

6

Change your admin PIN

In the file, find const ADMIN_PIN and change '1234' to your preferred PIN. Only you will know it.

7

Upload to GitHub Pages

Push the file to a GitHub repo → Settings → Pages → Deploy from main branch. Your URL will be yourusername.github.io/repo-name. Share it with all staff!

Until Firebase is configured: The app uses browser localStorage — it works for one device only (great for testing). Once Firebase is set up, all devices share the same live deck.
